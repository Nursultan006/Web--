В этой части книги рассматривается обзор языка Фортран, основные команды и структура программы.

1. МЕТАКОМАНДЫ МС-ФОРТРАНА

Метакоманды позволяют управлять компиляцией и запуском программы.
Примеры:
- $INCLUDE
- $DEFINE

2. ОСНОВЫ ЯЗЫКА

2.1. Переменные и константы

Переменные объявляются с указанием типа:
INTEGER :: a, b
REAL :: x, y

2.2. Подпрограмма

Подпрограмма — это блок кода, который можно вызвать из основной программы.
Пример:
SUBROUTINE Sum(a,b)
  INTEGER :: a,b
  ...
END SUBROUTINE Sum
